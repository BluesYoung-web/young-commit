# 说明

## 安装

```bash
npm i young-commit -g
```

## 基础使用

- 将修改添加到暂存区之后
- 命令行输入 `young-commit` 或者别名 `yc`

## 带参的快捷操作

```bash
USAGE young-commit [OPTIONS] 
# USAGE yc [OPTIONS] 

OPTIONS

     -i, --init    首次提交
  -r, --release    版本发布
```
